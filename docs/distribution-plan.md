# toddleSpot — Distribution Plan

> Goal: Drive traffic to toddlespot.co.uk and validate demand via waitlist signups.
> Budget: £0. All free channels.
> Focus area: Upminster and surrounding Essex area.

---

## Tactics

### Week 1 — Immediate

#### 1. Local Facebook Groups (Highest Impact)
- Find and join Upminster/Hornchurch/Essex parent groups
- Post conversationally — ask for input, don't sell
- Angle: *"I'm a local dad building an app to help parents find baby-friendly venues — would this be useful to you?"*
- Aim for 2-3 groups minimum

#### 2. Personal WhatsApp Network
- Share with every parent you know personally
- Ask them to forward to their parent WhatsApp groups
- Personal recommendation converts better than any other channel

#### 3. Instagram
- Create a toddlespot Instagram account
- Post 3-4 relatable parent pain point reels/stories (doesn't need to be polished)
- Use local hashtags: #upminster #upminstermums #essexparents #babydays #mumlife
- Link in bio → toddlespot.co.uk

### Week 2 — Expand

#### 4. Mumsnet Local
- Post in the Mumsnet Local talk board for the Upminster/Havering area
- Same conversational angle as Facebook — seeking input, not advertising

#### 5. Local Venues
- Visit baby-friendly cafés and pubs in Upminster in person
- Explain what you're building — ask if they'd share with parent customers
- Some may post your link on their socials or display a QR code

---

## Success Metrics (2-week target)

| Metric | Target |
|--------|--------|
| Landing page visits | 100+ |
| Waitlist signups | 10-20 |
| Conversion rate | 10-20% |

---

## How to Read the Results

- **10+ signups from cold traffic** → Genuine signal. Proceed with MVP.
- **100+ visits but 0 signups** → Messaging or value prop needs work. Iterate the landing page before building.
- **Can't get 100 visits** → Distribution problem, not product problem. Try different channels or angles.
